# App de Estudos - Ambiente Acadêmico Virtual

**Equipe:** Henry Clark e Maria Clara

## 📝 Situação-Problema
Alunos frequentemente possuem dificuldades em visualizar e organizar suas entregas de atividades, muitas vezes perdendo prazos por não encontrar o local de envio de forma rápida e intuitiva. O projeto foca em centralizar o acesso acadêmico para reduzir a fricção na rotina de estudos.

## 🎯 Tarefa principal do usuário
Realizar a entrega de uma atividade pendente dentro do prazo.

## 🗺️ Fluxo e Arquitetura da Informação
O usuário tem **dois caminhos distintos** para realizar a mesma tarefa principal:

* **Caminho 1 (Via Menu de Disciplinas):**
  Tela Inicial ➔ Ambiente Virtual ➔ Selecionar Disciplina desejada ➔ Selecionar Atividade Pendente ➔ Anexar arquivo ➔ Enviar.

* **Caminho 2 (Via Calendário):**
  Tela Inicial ➔ Calendário ➔ Selecionar a data da entrega (marcado por cores) ➔ Selecionar Atividade Pendente ➔ Anexar arquivo ➔ Enviar.
